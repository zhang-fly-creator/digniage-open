# Contributing

Thank you for helping improve DigniAge · 知老.

## How to contribute

- Open an Issue to report bugs, discuss risks, or propose improvements.
- Open a Pull Request with a clear description of the problem, the change, and how you tested it.
- Prefer small, focused changes.

## Please do NOT submit

- Real elder data
- Real phone numbers
- National ID numbers
- Home addresses
- Real service records
- Database backups
- API keys
- `.env` files
- Internal handoff documents

## Contribution areas we welcome

- UI improvements
- Accessibility
- Multilingual translation
- Demo data refinement
- Privacy templates
- AI safety prompts
- Supabase demo improvements

## Safety first

If your contribution touches elder-related content, please keep all examples fictional, de-identified, and suitable for public open-source review.

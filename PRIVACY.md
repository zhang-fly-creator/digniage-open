# Privacy Notice

This repository itself does not collect user data.

## English

- Demo data in this repository must remain fictional.
- Do not enter national ID numbers, detailed medical histories, family conflicts, asset information, home addresses, or real phone numbers into demo environments.
- Real deployments must obtain proper authorization, run in secure environments, and comply with local privacy and data protection laws.
- Any public-facing output must be properly de-identified before sharing.

## 中文说明

- 本开源仓库本身不收集用户数据。
- Demo 数据必须全部虚构。
- 不应录入身份证号、详细病历、家庭纠纷、财产信息、住址、真实手机号等敏感信息。
- 真实机构部署需自行取得授权、配置安全环境，并遵守当地法律法规。
- 对外展示内容必须先完成脱敏处理。
